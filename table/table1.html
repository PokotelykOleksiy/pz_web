Мій електронний довідник HTML
Призначення Назва тегу Короткий опис
Для каркасу документа <!DOCTYPE> Оголошує тип документа ы надаэ основну ынформацыю для браузера
<html> Вказує мову
<head> Для додаткової інформації
<title> Назва сторінки
Для роботи з текстом <p> Новий рядок
<h1><h2><h3><h4><h5><h6> Вказує заголовок і його розмір
<b><strong> Виділяє жирним шрифтом
<i> Курсив
<u> Підкреслення
<ol> Нумеровані списки
<li> Для окремого позначення нумерованого списку
<ul> Ненумеровані списки
<pre> Щоб текс який був оформлений у текстових редакторах не змінювався
<hr> Горизонтальна лінія
<br> відступ
Для створення таблиць <table> Створює таблицю
<tr> Нова строка
<td> Для запису у таблиці
Для гіперпосилань <a> Створює гіперпосилаючі силки
<img> Підключає зображення
Для підключення класів <div> Підключає клас
<var> Виділяє змінні з програм, відображаючи їх курсивом
Для роботи <source> Вказує місце розташування і тип альтернативних медіа ресурсів для елементів video та audio
<span> Контейнер для малих елементів. Можна використовувати для форматування уривків тексту, наприклад, виділення кольором окремих слів.
<style> Подключает встраиваемые таблицы стилей
<textarea> Створює великі поля для введення тексту.
<thead> Визначає заголовок таблиці.
